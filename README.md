 Highest Bidder / Auction Bot

A simple Python console program to run a silent auction and determine the highest bidder.

 Features
- Collects multiple bidders' names and bid amounts
- Determines the highest bidder automatically
- Works entirely in the console
- Beginner-friendly Python code

 How to Use
1. Run the program using Python 3.
2. Enter your name and bid amount when prompted.
3. When all bidders have entered their bids, type "no" to finish.
4. The program will display the winner and their bid.
